# Acme

A secure Flutter + .NET workspace generated by [ctx.0](https://github.com/).

| Dir | Stack |
|-----|-------|
| `app/` | Flutter (Bloc state management) |
| `api/` | .NET (Clean Architecture, EF Core code-first, PostgreSQL) |

Bundle id: `com.acme.acme`

## Getting started

```bash
docker compose up -d        # local PostgreSQL
(cd api && dotnet run)      # start the API
(cd app && flutter run)     # start the app
```

## Managing features

```bash
ctx0 status                 # what's enabled
ctx0 enable <feature>       # toggle a feature on (both sides)
ctx0 disable <feature>
ctx0 doctor                 # verify integrity + wire-protocol lock
```

The security plane (RASP, ALE, request signing, JWT, RLS, envelope encryption,
rate limiting) is vendored source under `app/lib/security/` and
`api/src/Security/`. Point your coding agent at `AGENTS.md`.
